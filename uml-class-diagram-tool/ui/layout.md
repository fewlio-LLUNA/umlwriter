# 全体構成

```
┌─────────────────────────────────────────────┐
│ Toolbar （上部・横一列）                       │
├──────────────────────────────┬──────────────┤
│                              │              │
│   Canvas （中央・作図領域）     │  Inspector   │
│                              │  （右・編集）  │
│                              │              │
└──────────────────────────────┴──────────────┘
```

- [Toolbar](toolbar.md): クラス追加・関連モード・入出力・画像書き出し
- [Canvas](canvas.md): React Flow の作図面。ドラッグ・ズーム・パン
- [Inspector](inspector.md): 選択中の要素の編集フォーム。未選択時は空状態の案内

# 振る舞い

- 要素未選択ではインスペクタは「クラスを選んで編集」等の空状態。
- キャンバスでクラス/関連を選ぶとインスペクタが切り替わる。
- 右ペインは折りたたみ可（任意）。

# Design への依頼

この 3 ペインを基点に、配色・余白・タイポグラフィ・空状態の見せ方を詰める。
クラスの箱と 6 種の関連線の見た目は [Canvas](canvas.md) / [Relationships](/uml-notation/relationships.md) を参照。
